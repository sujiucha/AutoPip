# AutoPip Installer Pro

用于 MCDReforged 的专业级 Python 依赖包管理与环境管家。

> **Important**
> 本插件旨在彻底解决 MCDR 服主在安装复杂插件时遇到的“缺少依赖”、“环境污染”、“包版本冲突”以及“运维无日志排错困难”等痛点。
> 强烈建议将其作为服务器的基石工具长期运行。

---

## 核心功能介绍

**安全隔离与预警**
- [x] 强制绑定宿主 Python 环境（或 venv）安装，绝不污染全局环境，具备多线程防死锁保护。
- [x] **依赖冲突预警**：底层引擎执行对账时，若发现多款插件对同一依赖版本要求存在冲突，将实施红字拦截警告，杜绝环境崩溃。

**智能守护与快照**
- [x] 透视扫描引擎：支持穿透读取 `.mcdr` 和 `.pyz` 压缩包内部的依赖配置。
- [x] **开机静默守护**：服务器启动时自动对账。若发现依赖缺失，延时在控制台高亮报警。
- [x] **环境快照体系**：提供对当前 Python 库的一键导出备份功能，支持全自动导入恢复，方便服务器异地迁移。

**依赖包与日志分析**
- [x] 支持在终端进行单点库文件的安装、强制卸载与一键升级检测。
- [x] **本地日志落地**：执行任何安装操作时，自动将 pip 运行流水输出至 `logs/autopip_日期.log`，方便事后提交排错报告。

**交互与热更新**
- [x] 智能分流输出，绕过原版客户端聊天栏点击安全拦截。
- [x] 全自动热更新：一键连线 GitHub 检查新版本，并全自动覆写升级自身代码。

## 指令字典速查

> 需要权限等级 3。也可直接输入 `!!pip` 呼出交互菜单。

**基础扫描与快照**
```text
!!pip check    -> 对账列出真正缺少的依赖，若发现插件间存在版本冲突将强制预警
!!pip install  -> 集中安装所有缺失包，并生成本地运行日志
!!pip freeze   -> 导出当前 Python 环境的所有库信息作为快照备份
!!pip restore  -> 读取快照备份，一键重建服务器依赖环境
```

**库文件精确管理**
```text
!!pip install <包名>    -> 手动安装 (例如: !!pip install requests)
!!pip uninstall <包名>  -> 强制卸载 (例如: !!pip uninstall emoji)
!!pip outdated          -> 检查所有环境包的最新可用版本
!!pip upgrade <包名>    -> 安全升级
```

**维护与运维**
```text
!!pip list [搜索词]     -> 查看库。控制台输出完整清单，游戏内只展示部分并提供搜索
!!pip update            -> 连接 GitHub，全自动热重载自身至最新版
!!pip reload            -> 重载配置文件
```
